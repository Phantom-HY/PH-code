### 题目描述

现在给你$n$个字符串

然后再给一个长度为$m$的字符串$S$

对于$S$的每个前缀你需要输出该前缀在$n$个字符串中作为前缀的次数
并换行

### 输入格式
第一行一个数字$n$，代表$n$个字符串

然后$n$行，每行一个字符串

然后一个长度为$m$的字符串$S$

### 输出格式

输出$m$行（要求见题目描述）

### 输入样例

```
3
apple abc apb
apc
```

### 输出样例
```
3
2
0
```
### 数据规模

对于$25\%$的数据，$ n,\sum |s| \leq 10$

对于$50\%$的数据，$n ，\sum |s| \leq 500$

对于$75\%$的数据，$n ，\sum |s| \leq 10000$

对于$100\%$的数据，$n ，\sum |s| \leq 100000$
