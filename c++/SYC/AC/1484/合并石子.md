### 题目描述
在一个操场上一排地摆放着$N$堆石子,现要将石子有次序地合并成一堆
规定每次只能选相邻的$2$堆石子合并成新的一堆，并将新的一堆石子数记为该次合并的得分
计算出将$N$堆石子合并成一堆的最小得分
### 输入格式
第一行为一个正整数$N (2 \leq N \leq 100)$

以下$N$行,每行一个正整数，小于$10000$，分别表示第$i$堆石子的个数$(1 \leq i \leq N)$

### 输出格式
一个正整数,即最小得分
### 输入样例1
```
3
1
2
3
```
### 输出样例1
```
9
```
### 输入样例2
```
7
13
7
8
16
21
4
18
```
### 输出样例2
```
239
```
