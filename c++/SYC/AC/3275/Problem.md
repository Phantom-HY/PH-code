###题目描述
仅由$8$或$9$组成的数深受大家的喜爱

将其从小到大排列如下

$8,9,88,89,98,99,888,889,898,899,988,989,998,999\cdots$

现在请你求出第$N$小的仅由仅由$8$或$9$组成的数

###输入格式
第一行输入一个正整数$N$

###输出格式
输出一个整数表示结果

###输入样例1
```
1
```
###输出样例1
```
8
```
###输入样例2
```
7
```
###输出样例2
```
888
```
###输入样例3
```
47
```
###输出样例3
```
98888
```
###数据规模
对于$10\%$的数据$1 \leq N \leq 100$

对于$45\%$的数据$1 \leq N \leq 2 \times 10^5$

对于$100\%$的数据$1 \leq N \leq 10^{18}$
