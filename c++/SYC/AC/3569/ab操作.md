### 题目描述
有三个整数 $a,b$ 与 $n$

每一次可以执行以下操作之一：

- 将 $a$ 增加 $b$

- 将 $b$ 增加 $a$

求使得 $a$ 和 $b$ 中**至少**有一个数**严格大于** $n$ **至少**需要操作几次

###  输入格式
输入三个整数 $a,b,n$
###  输出格式
输出一个整数表示最小操作次数
###  输入样例
```
1 2 3
```
###  输出样例
```
2
```
###  数据范围
对于 $50\%$ 的数据 $1 \leq a,b \leq n \leq 10^9$

对于全部的数据 $1 \leq a,b \leq n \leq 10^{16}$
