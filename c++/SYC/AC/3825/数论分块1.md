### 题目描述
定义 $f(i)$ 为 $i$ 的正约数个数

给出正整数 $N$ 请你求出

```katex
\displaystyle \sum_{i=1}^{N} f(i)
```

### 输入格式

输入一个整数 $N$

### 输出格式

输出一个整数表示 $\displaystyle \sum_{i=1}^{N} f(i)$

### 输入样例
```
5
```
### 输出样例
```
10
```
### 数据规模
对于 $10\%$ 的数据 $1 \leq N \leq 10^5$

对于 $30\%$ 的数据 $1 \leq N \leq 2 \times 10^7$

对于 $100\%$ 的数据 $1 \leq N \leq 10^{12}$
