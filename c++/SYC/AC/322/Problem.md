###  题目描述
给你一个$n$个数的数组，逆序对定义为:

存在两个整数$i \lt j$使得$a_i \gt a_j$

你需要求出逆序对的个数

###  输入
第一行$n$，代表数组长度
第二行$n$个数，代表$a_i (1 \leq i \leq n)$

###  输出

输出逆序对数


###  输入样例

```
4
4 3 2 1
```

###  输出样例

```
6
```

###  数据规模
对于$10\%$的数据，$n \leq 100, 1 \leq a_i \leq 100$
对于$40\%$的数据, $n \leq 10000, 1 \leq a_i \leq  10^9$
对于$100\%$的数据，$n \leq 100000, 1 \leq a_i \leq  10^9$
