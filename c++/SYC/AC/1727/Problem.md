### 题目描述
有一个 $3 \times 3$ 的平面魔方

在平面魔方中,每个格子里分别无重复地写上 $1 \sim 9$ 这 $9$ 个数字

一共有 $4$ 种对平面魔方的操作：

+ 选择某一行左移

+ 选择某一行右移

+ 选择某一列上移

+ 选择某一列下移

初始状态为

```
123
456
789
```

如选择第一行左移,魔方会变成下面这样

```
231
456
789
```
现在给出魔方的一个状态,问你能否将魔方复原成初始状态

如果可以,计算最少操作次数

### 输入格式
输入三行,每行三个 $1$ 到 $9$ 之间的整数
### 输出格式
如果能还原成初始状态,输出最小的操作次数,否则输出  $-1 $
### 样例输入
```
412
756
389
```
### 样例输出
```
2
```
