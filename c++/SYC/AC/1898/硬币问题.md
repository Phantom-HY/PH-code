### 题目描述
有$n$种硬币,面值分别为$v_1, v_2, \dots, v_n$,每种都有无限多

给定非负整数$S$,可以选用多少个硬币,使得面值之和恰好为$S$？

输出硬币数目的最小值

### 输入格式
第一行两个整数,$n$,$S(1 \leq n \leq 100, 0 \leq S \leq 100000)$
第二行$n$个整数$v_i(1 \leq v_i \leq S)$

### 输出格式
一个整数,表示硬币数目的最小值 $a$ ,无解则输出 $-1$


### 输入样例
```
6 666
1 5 10 20 50 100
```
### 输出样例
```
10
```
