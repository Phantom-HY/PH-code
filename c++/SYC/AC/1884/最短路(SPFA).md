### 题目描述
给定 $M$ 条边, $N$ 个点的带权无向图

求 $1$到 $N$ 的最短路
### 输入格式
第一行输入两个整数$N,M$

接下来$M$行$3$个正整数：$a_i,b_i,c_i$表示$a_i,b_i$之间有一条长度为$c_i$的路,$c_i \leq 1000$

### 输出格式
输出一个整数,表示 $1$ 到 $N$ 的最短距离
### 输入样例
```
4 4
1 2 1
2 3 1
3 4 1
2 4 1
```
### 输出样例
```
2
```
### 提示
注意图中可能有重边和自环,数据保证 $1$ 到 $N$ 有路径相连
### 数据规模
对于全部的数据$N \leq 100000,M \leq 500000$
