###  题目描述
假设用一个 $n \times n$ 的数组 $a$ 来描述一个有向图的邻接矩阵：

（1）编写一个函数确定一个顶点的出度

（2）编写一个函数确定一个顶点的入度

（3）编写一个函数确定图中边的数目

###  输入格式
第一行：节点总数 $n$，指定节点 $m$。

下面 $n$ 行：有向图的邻接矩阵，相邻两数之间以一个空格分隔。

###  输出格式
第一行包括三个数据：节点编号 $m$，$m$ 的出度，$m$ 的入度（之间用一个空格隔开）。

第二行包括一个数据：图中边的总数。

###  样例输入
```
5 3
0 4 2 2 3
2 0 1 5 10
2 0 0 4 0
0 3 7 0 7
6 2 0 0 0
```
### 样例输出
```
3 2 3
15
```

###  数据范围
对于全部的数据$1\le n,m,a_{ij} \le 1000$
