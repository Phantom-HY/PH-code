###  题目描述

有一个经典的问题需要让你帮忙，就是区间合并，两个区间能合并成一个区间，当且仅当区间相邻或者有交集。相邻：[1,2], [2,4] -> [1,4]    交集：[1, 3], [2, 5] -> [1,5]

###  输入

第一行一个整数n

后面n行，每行2个数，代表区间




###   输出
输出若干行表示合并后的结果，按区间前后顺序输出



###  样例
```
3
1 2
2 4
5 5
```

```
1 4
5 5
```

```
2
1 3
2 2
```

```
1 3
```





###  数据范围

对于40%的数据，$1<=n<=100，1<=区间左右端点<=100$

对于100%的数据，$1<=n<=100000，1<=区间左右端点<=INT\\_MAX$
