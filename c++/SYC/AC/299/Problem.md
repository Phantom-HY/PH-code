### 题目描述
$Mas$对数字非常感兴趣,他最近在研究一种数叫$Mas$数
这种数不能被$2$,$3$,$5$之外的素数整除,只能被这$3$个素数整除

显然前几个$Mas$数为,$2,3,4,5,6,8,\cdots$

现在需要你输出第$n$个$Mas$数

### 输入格式
第一行一个数字$n$
### 输出格式
输出第$n$个$Mas$数
### 输入样例
```
7
```
### 输出样例
```
9
```
### 数据规模
对于$25\%$的数据,$n \leq 10$
对于$50\%$的数据,$n \leq 100$
对于$75\%$的数据,$n \leq 1000$
对于$100\%$的数据,$n \leq 10000$
