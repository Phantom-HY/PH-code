### 题目描述
昨天$Mas$在纸上写下了一个奇怪的函数:
```katex
f(n) = \sin(1+\sin(2+\sin(3\dots\sin(n)\dots)))
```
今天$Mas$又写下了一个新的函数
```katex
g(n)=(...(((f(1) + n)f(2) + n−1)f(3) +n−2)...)f(n)+1
```
### 输入格式
一个正整数$n(1 \leq n \leq 50)$
### 输出格式
输出这个函数

### 输出样例1
```
1
```
### 输出样例1
```
sin(1)+1
```

### 输出样例2
```
3
```
### 输出样例2
```
((sin(1)+3)sin(1+sin(2))+2)sin(1+sin(2+sin(3)))+1
```
